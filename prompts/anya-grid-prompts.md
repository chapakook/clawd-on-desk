# Anya Theme — 4×4 그리드 ChatGPT 프롬프트 (GIF별)

> **사용법**
> 1. ChatGPT Images 2.0에 아냐 참조 이미지 첨부
> 2. GIF별 프롬프트 복붙 → 1024×1024 그리드 생성
> 3. ffmpeg으로 16칸 크롭 → frame1.png ~ frame16.png
> 4. GIF 조립 → `themes/anya/assets/<파일명>.gif`
> 5. 검증: `node scripts/validate-theme.js ./themes/anya`
>
> **총 ChatGPT 호출:** 15번 | **GIF당 프레임:** 16

---

## 크롭 명령어 템플릿

1024×1024 이미지, 각 칸 256×256:

```bash
# FILENAME 을 실제 파일명으로 교체 후 실행
IMG=FILENAME_grid.png
for row in 0 1 2 3; do
  for col in 0 1 2 3; do
    n=$((row * 4 + col + 1))
    ffmpeg -i $IMG -vf "crop=256:256:$((col*256)):$((row*256))" frame${n}.png
  done
done
# GIF 조립 (8fps = delay 12)
ffmpeg -framerate 8 -i frame%d.png FILENAME.gif
```

- 얼굴형/헤어/눈 색상/의상 16칸 전체 100% 동일
- 칸 간 포즈 변화는 연속적·점진적 (애니메이션 프레임처럼)
