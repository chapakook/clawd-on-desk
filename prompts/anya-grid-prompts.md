# Anya Theme — 4×2 그리드 ChatGPT 프롬프트 (GIF별)

> **사용법**
> 1. ChatGPT Images 2.0에 아냐 참조 이미지 첨부
> 2. GIF별 프롬프트 복붙 → **가로형(1792×1024)** 선택 후 생성
> 3. ffmpeg으로 8칸 크롭 → frame1.png ~ frame8.png
> 4. 배경 제거: `rembg i frame1.png frame1.png` 또는 remove.bg 사용
> 5. GIF 조립 → `themes/anya/assets/<파일명>.gif`
> 6. 검증: `node scripts/validate-theme.js ./themes/anya`
>
> **주의:** ChatGPT는 투명 배경을 지원하지 않음 → 밝은 초록(#00FF00) 배경으로 생성 후 배경 제거 필요
>
> **총 ChatGPT 호출:** 15번 | **GIF당 프레임:** 8 | **이미지:** 1792×1024 | **셀 크기:** 448×512

---

## 크롭 명령어 템플릿

1792×1024 이미지, 4열 × 2행, 각 셀 448×512:

```bash
# FILENAME 을 실제 파일명으로 교체 후 실행
IMG=FILENAME_grid.png

# 1) 크롭
for row in 0 1; do
  for col in 0 1 2 3; do
    n=$((row * 4 + col + 1))
    ffmpeg -i $IMG -vf "crop=448:512:$((col*448)):$((row*512))" frame${n}.png
  done
done

# 2) 배경 제거 (rembg 설치: pip install rembg)
for n in 1 2 3 4 5 6 7 8; do
  rembg i frame${n}.png frame${n}.png
done

# 3) GIF 조립 (8fps = delay 12)
ffmpeg -framerate 8 -i frame%d.png FILENAME.gif
```

> **rembg 미사용 시:** https://www.remove.bg 에서 각 프레임 업로드 후 다운로드
