# Anya Theme — 8프레임 그리드 ChatGPT 프롬프트

> **사용법**
> 1. ChatGPT Images 2.0에 아냐 참조 이미지 첨부
> 2. 상태별 프롬프트 블록 복붙 → 2×4 그리드 이미지 생성
> 3. 생성된 이미지를 8칸으로 크롭 → 각 PNG 저장
> 4. ezgif.com 또는 ffmpeg으로 8프레임 GIF 조립 (delay: 12)
> 5. `themes/anya/assets/<파일명>.gif` 로 저장
> 6. 검증: `node scripts/validate-theme.js ./themes/anya`
>
> **총 ChatGPT 호출 횟수:** 15번 (상태 15개 × 1번씩)

---

## 공통 출력 사양 (모든 프롬프트에 동일 적용)

```
[출력 사양]
- 비율: 2:1 가로형, 해상도 1024×512
- 배치: 2행 × 4열 그리드, 8칸 (Frame 1~8, 좌상단→우하단 순)
- 배경: 순수 흰색(#FFFFFF), 칸 사이 여백 충분히
- 각 캐릭터: 얼굴+상반신, 흰색 외곽선(약 6px) 스티커 컷아웃

[스타일]
- 치비 애니메이션 일러스트, 2D, 파스텔 톤
- 8칸 모두 동일 캐릭터(핑크 투 갈래 머리, 초록 눈), 포즈만 변화
- 각 칸 우하단에 작게 "F1"~"F8" 프레임 번호 표시

[엄수 사항]
- 얼굴형/헤어/눈 색상/의상 8칸 전체 100% 동일
- 한 칸과 다음 칸의 포즈 변화는 작고 연속적으로 (애니메이션처럼)
```

---

## 크롭 명령어 템플릿

생성된 이미지가 1024×512일 때 (각 칸 256×256):

```bash
# FILENAME 을 실제 파일명으로 교체
ffmpeg -i FILENAME_grid.png -vf "crop=256:256:0:0"   frame1.png
ffmpeg -i FILENAME_grid.png -vf "crop=256:256:256:0" frame2.png
ffmpeg -i FILENAME_grid.png -vf "crop=256:256:512:0" frame3.png
ffmpeg -i FILENAME_grid.png -vf "crop=256:256:768:0" frame4.png
ffmpeg -i FILENAME_grid.png -vf "crop=256:256:0:256"   frame5.png
ffmpeg -i FILENAME_grid.png -vf "crop=256:256:256:256" frame6.png
ffmpeg -i FILENAME_grid.png -vf "crop=256:256:512:256" frame7.png
ffmpeg -i FILENAME_grid.png -vf "crop=256:256:768:256" frame8.png

# GIF 조립 (8fps)
ffmpeg -framerate 8 -i frame%d.png FILENAME.gif
```
