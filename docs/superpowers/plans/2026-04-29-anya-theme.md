# Anya Theme Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a fully working Clawd on Desk custom theme featuring Anya Forger (Spy x Family), with a ChatGPT Images 2.0 prompt kit for generating all animation state assets.

**Architecture:** Pull the user's fork into the current repo, generate the Anya theme scaffold via `create-theme.js`, configure `theme.json` for Anya, and write a prompt kit file. Assets are generated by the user using the prompts and placed into the theme folder; `validate-theme.js` verifies each addition.

**Tech Stack:** Node.js, Electron (Clawd on Desk), ChatGPT Images 2.0 (user-side), GIF assembly tool (ezgif.com or ffmpeg)

# Anya Theme — ChatGPT Images 2.0 Prompt Kit

## How to Use

1. Open ChatGPT with Images 2.0 enabled
2. Copy the BASE STYLE block below
3. Append the per-state variation for the image you need
4. Generate → download as PNG
5. For animated states: generate 4–6 frames with slightly varied poses, then assemble into GIF using https://ezgif.com/maker or `ffmpeg`
6. Place file in `themes/anya/assets/<state>.<ext>`
7. Run: `node scripts/validate-theme.js themes/anya`

---

Anya Forger from Spy x Family anime, chibi pixel art style,

## GIF Assembly (ffmpeg)

If you have multiple PNG frames named `frame1.png`, `frame2.png`, etc.:

```bash
ffmpeg -framerate 4 -i frame%d.png -vf "scale=200:200" output.gif
```

Or use https://ezgif.com/maker — upload frames, set delay to 25 (= 4fps), click "Make GIF".
